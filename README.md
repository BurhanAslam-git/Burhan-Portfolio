## Tech Stack

- **React** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework with custom design tokens
- **Framer Motion** - Animation library
- **React Icons** - Icon library

## Design Features

- **Glassmorphism**: Backdrop blur effects with transparency
- **Gradient Backgrounds**: Multi-color gradients for depth
- **Dark Mode**: Full theme support with toggle
- **Premium Styling**: Rounded corners, soft shadows, hover effects
- **Typography**: Inter (body) and Poppins (headings) from Google Fonts
- **Lighting Effects**: Radial gradients and subtle highlights

## Getting Started

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Customization

### Update Personal Information

1. Edit `src/data/projects.js` to add your projects
2. Edit `src/data/experience.js` to update your work experience
3. Edit `src/data/skills.js` to customize your skills
4. Update contact information in `src/components/Contact/Contact.jsx`
5. Update social media links in `src/components/Hero/Hero.jsx` and `src/components/Footer/Footer.jsx`

### Styling

The project uses Tailwind CSS. You can customize colors, fonts, and other design elements in `tailwind.config.js`.
